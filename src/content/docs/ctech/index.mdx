Bem-vindo à documentação do **CTech**.

## Visão Geral

O CTech é uma plataforma de tecnologia e inovação.

## Começando

### Pré-requisitos

- Node.js 18+
- PostgreSQL
- Docker (opcional)

### Instalação

```bash
git clone https://github.com/danieldcc/ctech
cd ctech
pnpm install
pnpm run dev
```
