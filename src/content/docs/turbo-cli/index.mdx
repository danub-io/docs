Welcome to the **turbo-cli** documentation вЂ” an inline terminal assistant (Claude Code style) with 4 agent modes (Normal, Plan, Code, Ask) for chatting with LLM models via API directly from the terminal.

<Card title="4 Agent Modes">
Normal (general assistant with bash), Plan (planning with project_inspector), Code (autonomous plan execution), Ask (general conversation + fetch).
</Card>

<Card title="LLM Client (OpenAI SDK)">
Client compatible with OpenAI, Groq, Together, Anthropic, and any provider that implements the OpenAI-compatible API.
</Card>

<Card title="Slash Commands">
Commands `/model`, `/key`, `/mode`, `/plan`, `/plan-menu`, `/plan-abort`, `/code`, `/code-reset`, `/code-status`, `/add`, `/clear`, `/new`, `/quit`, `/end`, `/help`.
</Card>

## Quick Start

```bash
# Clone
git clone <repo-url> turbo-cli
cd turbo-cli

# Create venv and install
python3 -m venv .venv
source .venv/bin/activate
pip install -e .

# Run (3 equivalent methods)
turbo
# python -m turbo_cli
# python main.py
```
