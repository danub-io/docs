Bem-vindo à documentação do **MBA Lite**.

## Visão Geral

Plataforma de ensino e cursos online com foco em administração.

## Começando

### Pré-requisitos

- Node.js 18+
- PostgreSQL
- Redis

### Instalação

```bash
git clone https://github.com/danieldcc/mbalite
cd mbalite
pnpm install
pnpm run dev
```
