Welcome to the **MBA Lite** documentation, the landing page for the *MBA Lite — Gestao para Alta Performance* course.

---

## Overview

Static landing page (SSG) built with Astro 6, Tailwind CSS v4, and shadcn/ui. Component-based design with independent sections, all pre-rendered.

### Stack

| Technology | Role |
|------------|------|
| **Astro 6** | SSG Framework |
| **React 19** | Islands of interactivity |
| **Tailwind CSS v4** | Styling with `@theme` |
| **shadcn/ui (new base) + Base UI** | UI components |
| **Geist Variable** | Typography |

## Page Components

| Component | Description |
|-----------|-------------|
| `Header` | Top navigation with anchor links |
| `Hero` | Main section with CTA |
| `Features` | Grid of course benefits |
| `LeadCapture` | Lead capture form |
| `AuthorBio` | Author biography |
| `Footer` | Footer with links |

## Getting Started

```bash
git clone <repo-url>
cd mbalite
pnpm install
pnpm dev          # Local server
```

### Build

```bash
pnpm build        # Generates dist/
pnpm preview      # Preview the build
```
