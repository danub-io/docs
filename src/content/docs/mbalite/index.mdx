Bem-vindo à documentação do **MBA Lite**, landing page do curso *MBA Lite — A Série Gestão para Alta Performance*.

---

## Visão Geral

Landing page estática (SSG) construída com Astro 6, Tailwind CSS v4 e shadcn/ui. Design componentizado com seções independentes, todas pré-renderizadas.

### Stack

| Tecnologia | Função |
|------------|--------|
| **Astro 6** | Framework SSG |
| **React 19** | Ilhas de interatividade |
| **Tailwind CSS v4** | Estilização com `@theme` |
| **shadcn/ui (base-nova) + Base UI** | Componentes de UI |
| **Geist Variable** | Tipografia |

## Componentes da Página

| Componente | Descrição |
|------------|-----------|
| `Header` | Navegação superior com links âncora |
| `Hero` | Seção principal com CTA |
| `Features` | Grid de benefícios do curso |
| `LeadCapture` | Formulário de captura de leads |
| `AuthorBio` | Biografia do autor |
| `Footer` | Rodapé com links |

## Começando

```bash
git clone <repo-url>
cd mbalite
pnpm install
pnpm dev          # Servidor local
```

### Build

```bash
pnpm build        # Gera dist/
pnpm preview      # Preview do build
```
