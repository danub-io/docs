## Visão Geral

Plataforma completa de EAD com gerenciamento de cursos, alunos, matriculas e certificados, com foco em administração e negócios.

## Arquitetura

```

                                           └─────────────┘
```

## Começando

### Pré-requisitos

- Node.js 22+
- PostgreSQL 16+
- Redis 7+
- pnpm

### Instalação

<Tabs>
  <TabItem label="Desenvolvimento">

    ```bash
    git clone https://github.com/danieldcc/mbalite
    cd mbalite
    pnpm install
    cp .env.example .env
    pnpm run db:push
    pnpm run dev
    ```

  </TabItem>
  <TabItem label="Docker (recomendado)">

    ```bash
    git clone https://github.com/danieldcc/mbalite
    cd mbalite
    cp .env.example .env
    docker compose up -d
    ```

  </TabItem>
</Tabs>

<Aside type="tip" title="Prisma Studio">
  Use `pnpm run db:studio` para visualizar e editar dados do banco via interface web.
</Aside>

### Configuração Inicial

<Steps>
  1. **Clone e instale** — `git clone` + `pnpm install`
  2. **Configure o banco** — ajuste `DATABASE_URL` no `.env`
  3. **Execute as migrations** — `pnpm run db:push`
  4. **Popule com dados iniciais** — `pnpm run db:seed`
  5. **Inicie o servidor** — `pnpm run dev`
</Steps>

<Aside type="caution" title="Redis">
  O Redis é usado para cache de sessão e filas. Sem ele, o login e a geração de certificados podem falhar.
</Aside>

## API

### Endpoints

| Método | Rota | Descrição |
|--------|------|-----------|
| `GET` | `/api/courses` | Lista cursos |
| `POST` | `/api/courses` | Cria curso |
| `GET` | `/api/courses/:id` | Detalhes do curso |
| `PUT` | `/api/courses/:id` | Atualiza curso |
| `DELETE` | `/api/courses/:id` | Remove curso |
| `POST` | `/api/enrollments` | Realiza matrícula |

## Deploy

```bash
# Build
pnpm run build

# Produção
pnpm run start

# Docker produção
docker compose -f docker-compose.prod.yml up -d
```

## FAQ

<Aside type="note" title="Erro de schema Prisma?">
  Execute `pnpm run db:push` para sincronizar o schema com o banco de dados.
</Aside>

<Aside type="danger" title="Upload de arquivos não funciona?">
  Verifique as configurações do MinIO no `.env` e se o serviço está rodando.
</Aside>
