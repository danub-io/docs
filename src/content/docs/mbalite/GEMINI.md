Projeto criado com Astro, Tailwind CSS (v4) e shadcn/ui.

## Tecnologias

- **Framework:** Astro
- **Estilização:** Tailwind CSS v4
- **Componentes:** shadcn/ui (Radix UI + React)
- **Fontes:** Inter, Work Sans (Google Fonts)

## Estrutura de Pastas

- `src/components/`: Componentes Astro e React (UI).
- `src/layouts/`: Layouts base do projeto.
- `src/pages/`: Páginas do projeto.
- `src/styles/`: Estilos globais e configuração do Tailwind v4.

## Customização do Design

O sistema de design foi configurado no arquivo `src/styles/global.css` usando as variáveis do `@theme` do Tailwind v4. As cores e fontes seguem a especificação fornecida.

## Como Executar

```bash
pnpm install
pnpm dev
```
