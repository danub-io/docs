Bem-vindo à documentação do **GospelReads**.

## Visão Geral

Aplicativo de leitura e estudos bíblicos com funcionalidades de comunidade.

## Começando

### Pré-requisitos

- Node.js 18+
- React Native CLI
- Firebase

### Instalação

```bash
git clone https://github.com/danieldcc/gospelreads
cd gospelreads
pnpm install
pnpm run start
```
