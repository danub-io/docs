## Overview

Static blog (SSG) built with Astro 6 and React 19, hosted on Firebase Hosting with aggressive caching.

```
Markdown (MD) → Content Collections → Astro Pages → Static HTML → Firebase Hosting
```

### Stack

| Technology | Role |
|------------|------|
| **Astro 6** | SSG Framework |
| **React 19** | Interactive components |
| **Tailwind CSS v4** | Styling + Typography |
| **Content Collections** | Content management |
| **Firebase Hosting** | Hosting with global CDN |
| **date-fns** | pt-BR date formatting |

## Content Structure

```
src/content/
├── posts/          # MD articles with frontmatter
├── authors/        # Author profiles
└── pages/          # Static pages
```

Posts have a schema with `title`, `description`, `date`, `authors`, `tags`, `image`.

## Getting Started

```bash
git clone <repo-url>
cd gospelreads
pnpm install
pnpm dev          # Local server (localhost:4321)
```

### Build and Deploy

```bash
pnpm build        # Generates dist/
firebase deploy   # Publishes to Firebase Hosting
```

## Performance

- 100/100 Lighthouse performance
- Pure SSG with no JavaScript on initial load
- 1-year cache for static assets
- Optimized images with CSS filters
