Bem-vindo à documentação do **GospelReads**, um blog de conteúdo teológico e reflexões cristãs com performance 100/100 Lighthouse.

---

## Visão Geral

Blog estático (SSG) construído com Astro 6 e React 19, hospedado no Firebase Hosting com cache agressivo.

```
Markdown (MD) → Content Collections → Páginas Astro → HTML estático → Firebase Hosting
```

### Stack

| Tecnologia | Função |
|------------|--------|
| **Astro 6** | Framework SSG |
| **React 19** | Componentes interativos |
| **Tailwind CSS v4** | Estilização + Typography |
| **Content Collections** | Gerenciamento de conteúdo |
| **Firebase Hosting** | Hospedagem com CDN global |
| **date-fns** | Formatação de datas pt-BR |

## Estrutura de Conteúdo

```
src/content/
├── posts/          # Artigos em MD com frontmatter
├── authors/        # Perfis de autores
└── pages/          # Páginas estáticas
```

Os posts possuem schema com `title`, `description`, `date`, `authors`, `tags`, `image`.

## Começando

```bash
git clone <repo-url>
cd gospelreads
pnpm install
pnpm dev          # Servidor local (localhost:4321)
```

### Build e Deploy

```bash
pnpm build        # Gera dist/
firebase deploy   # Publica no Firebase Hosting
```

## Performance

- 100/100 Lighthouse performance
- SSG puro sem JS no carregamento inicial
- Cache de 1 ano para assets estáticos
- Imagens otimizadas com filters CSS
