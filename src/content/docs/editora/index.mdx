Bem-vindo à documentação da **Editora**, uma ferramenta de vibe coding para produção profissional de livros. Escreva capítulos em Markdown e a Editora cuida da editoração — revisão com IA, diagramação e exportação para PDF (print-ready) e EPUB.

---

## Visão Geral

```
chapters/*.md → [IA: Edição/Revisão] → [Pandoc + LaTeX] → output/*.pdf / *.epub
```

### Stack Tecnológica

| Ferramenta | Função |
|------------|--------|
| **Python 3.12+** | Linguagem principal |
| **Typer** | Interface CLI |
| **Pydantic** | Validação e configurações |
| **Pandoc + LaTeX** | Diagramação profissional |
| **LangChain** | Orquestração de IA |
| **LanguageTool** | Gramática pt-BR |

## Funcionalidades

| Funcionalidade | Descrição |
|----------------|-----------|
| Escrita em Markdown | Capítulos versionáveis com Git e fáceis de converter |
| Edição com IA | Copyediting em 3 níveis (light/medium/aggressive) |
| Revisão de Consistência | Detecção de contradições, furos na trama e timeline |
| Proofreading | Correção gramatical e ortográfica pt-BR |
| Typesetting | PDF print-ready (KDP compliant) e EPUB |
| CLI Intuitiva | Comandos simples para todo o fluxo |

## Começando

### Pré-requisitos

- Python 3.12+
- Pandoc
- LaTeX (TeX Live ou TinyTeX)

### Instalação

```bash
git clone <repo-url>
cd editora
python -m venv .venv
source .venv/bin/activate
pip install -e .
```

### Uso Rápido

```bash
editora init --title "Meu Livro" --author "Seu Nome" --output meu-livro
cd meu-livro
# Escreva os capítulos em chapters/*.md
editora build    # Gera PDF + EPUB
```

## Comandos CLI

| Comando | Descrição |
|---------|-----------|
| `editora init` | Cria novo projeto de livro |
| `editora build` | Compila para PDF/EPUB |
| `editora edit` | Edita capítulos com IA |
| `editora proofread` | Revisa gramática e ortografia |
| `editora consistency` | Verifica consistência global |
| `editora template` | Gera templates LaTeX para formatação |
| `editora info` | Mostra estatísticas do projeto |
