Bem-vindo à documentação da **Editora**.

## Visão Geral

Sistema de gerenciamento editorial para publicação de livros.

## Começando

### Pré-requisitos

- Node.js 18+
- MongoDB
- AWS S3 (ou compatível)

### Instalação

```bash
git clone https://github.com/danieldcc/editora
cd editora
pnpm install
pnpm run dev
```
